## Arduino R4 Numpad Communication

# Funktion

Dieses kleine Script in C++ ermöglicht die Kommunikation mit einem Numpad und printet die Daten daten auf einem LCD Display. Wenn die eingabe = 1 ist wird auch dem LCD Display richtig angezeigt.

# Biblotheken

Biblotheck für das LCD Display: `#include <LiquidCrystal_I2C.h>`

Biblotheck für das Keypad: `#include "Adafruit_Keypad.h"`
